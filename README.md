# Home-Automation-System
## Home Automation System Using NODE MCU and relay module ( can be Controlled By Voice Command,web server & Android Application) 
# Things that you’ll need:
1.) NodeMCU
2.) Relay Board
3.) Jumper Cables
4.) Bread Board
5.) Smartphone.
